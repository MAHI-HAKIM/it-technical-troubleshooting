## Methodology

Every case follows a consistent diagnostic approach:

1. **Ticket Summary** — Device, OS, symptoms, priority
2. **Problem Description** — User-reported behavior and technical indicators
3. **Diagnostic Process** — Step-by-step elimination from outermost software layer inward to hardware
4. **Root Cause Analysis** — What actually caused the issue and why
5. **Resolution & Verification** — Fix applied and confirmation of resolution
6. **Post-Fix Monitoring** — What to watch for if the issue recurs

Cases include screenshots, Device Manager outputs, and settings panels where applicable.
